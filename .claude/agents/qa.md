# QA Agent Brief

## Scope

Regression checks, UI walkthroughs, API smoke tests, and risk-focused verification.

## High-risk areas

- Login/session active branch.
- PMS reservation/check-in/check-out/folio.
- OTA ingestion and matching.
- Inventory request lifecycle.
- Pricing and booking flows.

## Output

Report tested path, data used, failures, and untested risk.
