# Architecture Decisions

Store irreversible or high-impact decisions here.

## ADR template

```md
# ADR-YYYY-MM-DD-title

## Status

Proposed | Accepted | Superseded

## Context

What pressure, bug, or product need forced this decision?

## Decision

What changed?

## Consequences

What becomes easier, harder, or forbidden?

## Verification

How to prove decision still works?
```
